## **mini-builder (POC de motor de agentes con ADK)** 

Funcionalidades principales:
- **Gestión de agentes**: Crear, configurar y administrar agentes personalizados.
- **Manejo de sesiones**: Iniciar interacciones con agentes (chat, provisional)
- **Herramientas básicas para agentes**: RAG y web search
- **Ingesta de archivos**: PDFs, necesarios para RAG
- **Telemetría y trazabilidad**: Seguimiento de interacciones, uso de herramientas, rendimiento, etc. con Langfuse
